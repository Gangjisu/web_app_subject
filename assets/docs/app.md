# 화면
 ## 화면 구성 - 주요 기능: 사이트 첫 화면 또는 메인 대시보드에서 두 개의 대형 카드/버튼으로 선택지를 제공합니다.
  -
    
    새 계획 생성 카드/버튼: 지도 배경에 “새 여행 계획 만들기”라는 제목과 간단한 설명을 넣고, 클릭 시 위치 선택 화면으로 이동합니다.
    
    생성된 계획 카드/버튼: 현재 진행 중이거나 완료된 계획을 확인할 수 있는 버튼으로, 일정 개수나 상태(진행 중/예약 완료/완료)를 나타내는 작은 배지를 함께 표시해 사용자 관심을 끕니다.
    
    애니메이션: 버튼 위에 마우스를 올리면 새 계획 생성 카드/버튼은 지도가 움직여주고 그림자가 생기거나 살짝 움직여 상호작용을 유도합니다, 폴더를 열어보는 듯한 효과를 추가하고 그림자가 생기거나 살짝 움직여 상호작용을 유도합니다, 버튼을 누르고 나서는 페이지를 넘기는 듯한 부드러운 페이드 아웃 후 다음 화면으로 전환됩니다.


 ## 2-1 단계: 새 계획 생성 흐름

    1. 여행 위치 선택 화면
    

    2. AI 일정 추천 단계 화면

    프로그레스 단계: 선택된 위치가 정해지면 다음 화면에서는 가이드 메시지와 함께 AI 추천 프로세스가 진행됩니다. 단계별 진행률(예: 1/3 – 일정 추천, 2/3 – 숙소 추천, 3/3 – 교통 추천)을 표시하여 사용자가 현재 위치를 확인할 수 있게 합니다.

    일정 제안 카드: AI가 일정(예: “1일차 오전: 해변 산책, 오후: 지역 박물관 방문”)을 하나씩 제안하면, 사용자에게 ‘선택’ 또는 ‘수정’ 옵션을 제공합니다. 카드를 좌우로 스와이프하여 다음 제안을 볼 수 있는 방식으로 구성하면 게임처럼 즐겁게 느껴집니다.

    사용자 선택 반영: 사용자가 일정 요소를 선택할 때마다 우측 패널에 선택된 일정이 쌓여가는 모습을 시각적으로 보여줍니다. 선택하지 않은 경우 새 제안을 요청할 수 있도록 “다른 제안 보기” 버튼도 제공합니다.

    3. 계획 생성 및 숙소,교통 추천 화면

    요약 페이지: 일정 선택이 끝나면 다음 단계에서 전체 계획 요약을 보여주는 한 페이지 레이아웃을 구성합니다. 상단에는 여행 날짜와 여행지, 하루별 일정이 타임라인 형태로 펼쳐집니다.

    숙소 추천 섹션: 일정 바로 아래에는 추천 숙소 리스트를 카루셀 형태로 보여줍니다. 각 카드에는 숙소 사진, 등급, 가격, 위치, 주요 편의시설 아이콘 등이 포함되고, “최저가 보기” 버튼을 배치합니다.



    교통/항공 추천 섹션: 숙소 아래에는 항공권이나 교통수단(고속철도, 렌터카 등)을 검색할 수 있는 옵션을 제공합니다. 가장 저렴한 항공편/교통 옵션을 기본 추천으로 표시하고, 필터로 시간대·항공사·가격 범위를 변경할 수 있습니다.



    전체 예약 요약: 페이지 상단에 세션 선택 버튼으로 각 섹션으로 이동하기 쉽게 되어있고 페이지 하단에 모든 섹션에 공통 적으로 현재 선택한 숙소·교통 비용 합계와 일정 요약을 보여주어, 사용자가 최종 결정을 내릴 때 참고하기 쉽게 합니다. “계획 저장”과 “지금 예약” 두 개의 CTA 버튼을 명확하게 배치합니다.



    동적 효과: 각 섹션(일정 요약, 숙소 추천, 교통 추천) 사이 전환 시 슬라이드 애니메이션을 넣어 정보를 자연스럽게 연결합니다.
 ## 2-2. 기존 계획 보기 및 이어서 진행하기

    1. 대시보드 화면

    계획 카드: 카드를 생성된 순서대로 배치하고 계획 하나당 카드를 만들어 여행지 사진, 여행 제목(예: “2026 제주 가족 여행”), 여행 날짜 범위, 상태(예: 일정 선택 중 / 숙소 예약 완료 / 여행 완료) 라벨을 표시합니다. 진행 상태 라벨은 색상 과 아이콘(진행 중은 파란색, 완료는 회색 등)으로 직관적으로 구분합니다. 

    필터와 검색: 다수의 계획을 가진 사용자라면 계획명, 여행지, 기간 기준으로 검색하거나 정렬할 수 있는 옵션을 제공합니다.

    계속 진행 버튼: 진행 중인 계획 카드에는 ‘이어하기’ 또는 ‘계속 예약’ 버튼을 추가하여 자연스럽게 다음 단계로 넘어가도록 유도합니다.

    2. 세부 계획 관리

    편집 화면: ‘이어하기’ 버튼을 클릭하면 해당 상태에 맞는 2-1 페이지로 이동.

    5. 애니메이션과 상호작용 요소 제안

    페럴랙스 스크롤 효과: 지도 배경이나 여행지 이미지에 깊이감을 주는 페럴랙스 스크롤을 적용해 시각적 재미를 더합니다.

    로딩 애니메이션: AI가 일정을 계산하는 동안 여행과 관련된 아이콘(비행기, 지도 핀 등)이 회전하거나 움직이는 애니메이션으로 기다림을 덜 지루하게 합니다.

    카드 플립 효과: 카드형 정보(숙소, 일정 제안 등)를 클릭하면 뒷면에서 추가 정보가 나타나는 플립 애니메이션을 활용해 정보 탐색을 재미있게 만듭니다.

    미세한 진동/색상 변화: 버튼을 클릭하거나 항목을 선택할 때 미세한 진동(모바일)이나 색상 변화로   피드백을 제공해 사용자 참여를 높입니다.
